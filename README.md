### 远程控制 mac

````
ssh root@ip

````

### 安装ubuntu系统

````
apt-get update

````

### ssh连接到服务器

````
ssh root@ 服务器地址

````

### 安装node.js

````
apt-get install -y curl

curl -sl https://deb.nodesource.com/setup_4.x | sudo -E bash -

apt-get install nodejs

````

### 安装mysql

````
apt-get install mysql-server

apt-get install mysql-client

apt-get install libmysqlclient-dev
````
